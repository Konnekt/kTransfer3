Yippee! I've managed to get the examples to build under Eclipse.

Assuming you are already familiar with Eclipse and its CDT (C/C++ development tools) the steps to build one of the examples supplied with SmartWin++ are:

  • Create a new "Standard Make C++ Project"
  • Set the environment variable SMARTWINDIR to the directory containing your SmartWin++, there should not be a trailing slash
  • Import the source and resource files and the "Makefile.mingw.main"
  • Rename "Makefile.mingw.main" to "Makefile"
  • Rebuild the project
  • Run "main-mingw.exe"
